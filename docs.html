Introdução

O QuizLab é uma plataforma client-side para criação e execução de simulados. Esta documentação destina-se tanto a usuários finais quanto a desenvolvedores.

Guia do Usuário: Criador Visual

O assistente de criação funciona em etapas para garantir a integridade dos dados:

  1. Dados Gerais: Defina o título. O botão de confirmação libera a edição de questões.
  2. Questões: Preencha enunciado e alternativas. Marque as corretas usando o checkbox lateral.
  3. Ocultar/Exibir: Clique no cabeçalho de uma seção para recolhê-la e ganhar espaço.
  4. Exportação: Use o botão de download para salvar seu arquivo .JSON.

Guia do Desenvolvedor: Estrutura JSON

Esquema de dados para importação manual:

Esquema Raiz

{
  "nomeSimulado": "String",
  "descricao": "String (Opcional)",
  "questoes": [ QuestionObject ]
}

Objeto de Questão

{
  "id": Number,
  "enunciado": "String",
  "tipo": "unica" | "multipla",
  "alternativas": [
    { "id": "a", "texto": "String" }
  ],
  "respostasCorretas": ["a"] 
}
