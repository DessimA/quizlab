Início Rápido

Comece a usar o QuizLab em menos de 1 minuto. Não precisa de cadastro.

Opção A Importar um simulado pronto

Baixe o arquivo de exemplo (.json), clique em "Começar Agora" e arraste o arquivo para a área de upload.

Opção B Criar do zero

Na tela inicial, clique em "Criar Simulado". Preencha o título, adicione questões com suas alternativas e marque a resposta correta em cada uma.

Dica: Você não precisa de um arquivo para começar. O criador integrado faz tudo pelo navegador.

Modos de Jogo

Ao iniciar um simulado, você escolhe entre dois modos:

Modo Estudo

Após confirmar cada resposta, o sistema mostra imediatamente se você acertou ou errou, destacando a alternativa correta. Ideal para aprender e revisar conteúdo.

Modo Exame

Sem feedback durante o simulado igual a uma prova real. O resultado completo aparece apenas ao finalizar. Se o simulado tiver tempo limite definido, o cronômetro ativa automaticamente neste modo.

Embaralhamento: Ative nas opções de início para randomizar a ordem das questões e das alternativas a cada tentativa.

Criador de Simulados

O criador funciona em dois passos:

Passo 1 Dados gerais

Informe o título (obrigatório), descrição, tags de busca e, opcionalmente, um limite de tempo em minutos. Clique em "Confirmar" para avançar.

Passo 2 Questões

Cada questão precisa de um enunciado, tipo (única ou múltipla escolha), ao menos 2 alternativas e ao menos uma alternativa marcada como correta (checkbox). Use arrastar e soltar para reordenar.

Exportar e Salvar

Clique em "Exportar". Você pode salvar na biblioteca local (acesso rápido sem upload) ou baixar como arquivo .json para guardar no computador.

Atenção: Múltipla escolha exige que o usuário marque exatamente o número de respostas corretas que você definir. Defina 2 corretas apenas se a questão realmente tiver 2 respostas certas.

Formato do Arquivo JSON

Se preferir criar o arquivo manualmente ou integrar com outras ferramentas, use esta estrutura:

{
  "nomeSimulado": "Título do Simulado",
  "descricao": "Descrição opcional",
  "tags": ["matematica", "enem"],
  "tempoLimiteMinutos": 60,
  "questoes": [
    {
      "id": 1,
      "enunciado": "Qual a capital do Brasil?",
      "tipo": "unica",
      "alternativas": [
        { "id": "a", "texto": "São Paulo" },
        { "id": "b", "texto": "Brasília" },
        { "id": "c", "texto": "Rio de Janeiro" }
      ],
      "respostasCorretas": ["b"]
    }
  ]
}
Atenção: respostasCorretas deve sempre ser um array, mesmo para questões de única escolha. Os IDs das respostas devem coincidir exatamente com os IDs das alternativas.

Biblioteca Local

A biblioteca salva simulados diretamente no armazenamento do seu navegador (localStorage). Nenhum dado é enviado para servidores.

Acessar

Clique em "Meus Simulados" na tela inicial ou no header. Os simulados aparecem em cards com título, número de questões e sua média de acertos.

Editar

Cada card tem um botão de edição que abre o simulado no criador com todos os dados já preenchidos.

Atenção: Limpar o cache do navegador ou usar aba anônima apaga os dados. Sempre exporte simulados importantes como arquivo .json.

Perguntas Frequentes

O botão "Confirmar" não aparece

Em questões de múltipla escolha, você precisa selecionar exatamente o número de alternativas indicado no enunciado nem mais, nem menos.

Meu arquivo JSON dá erro

Verifique se o arquivo começa com { e termina com }, sem vírgulas sobrando. Use o JSONLint para validar a sintaxe.

Posso usar offline?

Sim. Após carregar a página uma vez, toda a lógica roda localmente. A única exceção é o carregamento das fontes do Google Fonts.

Como compartilhar um simulado?

Use o botão de download na biblioteca para exportar o arquivo .json e envie para outra pessoa. Ela importa da mesma forma que você.
