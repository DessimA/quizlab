Bem-vindo ao QuizLab

O QuizLab é uma ferramenta profissional para criação, gerenciamento e execução de simulados. Projetado com foco em privacidade e performance, ele funciona 100% no seu navegador, sem enviar dados para servidores externos.

📚 Biblioteca & Armazenamento

O QuizLab possui um sistema de cache inteligente para facilitar seus estudos:

  • Armazenamento Local: Seus simulados são salvos na memória do navegador (LocalStorage).
  • Limite de 10 Slots: Para garantir performance, você pode manter até 10 simulados salvos simultaneamente.
  • Gestão Automática: Ao tentar salvar o 11º simulado, o sistema oferecerá a opção de substituir o mais antigo ou exportá-lo antes da remoção.
  • Persistência: Seus dados permanecem salvos mesmo se você fechar a aba (desde que não limpe o cache do navegador).

📝 Realizando Simulados

Entenda como funciona o motor de provas:

  • Modo Foco: Em computadores, a barra de navegação se oculta automaticamente ao rolar para baixo para evitar distrações.
  • Navegação Mobile: Toque duas vezes rapidamente (Double Tap) em qualquer área vazia da tela para ocultar/exibir o menu superior.
  • Regras de Seleção:
    • Única Escolha: Selecione uma opção e clique em "Confirmar".
    • Múltipla Escolha: Você deve selecionar exatamente o número de respostas corretas indicado (ex: "Selecione 2 opções"). O botão de confirmação só aparecerá quando a quantidade estiver correta.
  • Revisão: Ao finalizar, você terá acesso a um relatório detalhado de erros e acertos, com opção de reiniciar o teste imediatamente.

🛠️ Criador de Simulados (Wizard)

Crie seus próprios testes sem precisar editar códigos:

  1. Dados Gerais: Defina um Título e Descrição. Clique no botão de confirmação para travar esses dados e iniciar a edição das questões.
  2. Adicionando Questões: O sistema utiliza um formato de "Acordeão". Ao adicionar uma nova questão, a anterior é validada e minimizada automaticamente para organizar a tela.
  3. Validação em Tempo Real: O botão "Nova Questão" só é habilitado se a questão atual estiver completa (Enunciado + Alternativas + Resposta Correta marcada).
  4. Exportação: Ao terminar, clique em "Exportar JSON". O arquivo será baixado e também salvo automaticamente na sua Biblioteca Local (se houver espaço).

💻 Guia para Desenvolvedores (JSON)

Se preferir criar seus simulados manualmente via editor de código, siga rigorosamente este esquema:

Estrutura do Arquivo

{
  "nomeSimulado": "Título do Simulado (Obrigatório)",
  "descricao": "Descrição breve (Opcional)",
  "questoes": [
    {
      "id": 1,
      "enunciado": "Texto da pergunta (Aceita HTML básico)",
      "tipo": "unica",  // ou "multipla"
      "alternativas": [
        { "id": "a", "texto": "Opção A" },
        { "id": "b", "texto": "Opção B" }
      ],
      "respostasCorretas": ["a"] // Array com IDs das corretas
    }
  ]
}

Regras de Validação de Dados

  • Tipo "unica": O array respostasCorretas deve conter exatamente 1 ID.
  • Tipo "multipla": O array respostasCorretas deve conter 2 ou mais IDs.
  • IDs: Os IDs dentro de respostasCorretas devem corresponder exatamente aos IDs definidos nas alternativas.

❓ Perguntas Frequentes (FAQ)

P: Meus simulados ficam salvos na nuvem?
R: Não. O QuizLab é "Zero Data". Tudo fica gravado apenas no seu navegador atual. Se você limpar o cache ou usar navegação anônima, os dados serão perdidos.

P: Posso usar no celular?
R: Sim. A interface é 100% responsiva e possui recursos exclusivos para mobile, como o "Modo Compacto" e controle por gestos.

P: O arquivo não carrega. O que fazer?
R: Verifique se o arquivo está no formato .json e se segue a estrutura descrita acima. Erros de vírgula ou aspas no JSON são as causas mais comuns.
