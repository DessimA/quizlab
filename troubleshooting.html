SOLUÇÃO DE PROBLEMAS

Diagnóstico e resolução para erros comuns

Problemas de Upload e Arquivo

Erro: "Arquivo não é JSON válido"

Causa Provável: O arquivo contém erros de sintaxe ou a extensão não é .json.

Solução:

  1. Abra o arquivo no Bloco de Notas ou VS Code.
  2. Verifique se começa com { e termina com }.
  3. Cole o conteúdo no site JSONLint para validar.

Erro: "Campo Obrigatório Ausente"

Causa Provável: O JSON não possui o campo nomeSimulado ou questoes.

Solução: Baixe o Arquivo de Exemplo e compare a estrutura.

Problemas Durante o Quiz

Botão "Confirmar" não habilita

Causa: Em questões de múltipla escolha, você selecionou mais ou menos opções do que o exigido.

Solução: Leia atentamente o aviso "(SELECIONE EXATAMENTE X)". Se pedir 2, marque apenas 2.

Travou ou Tela Branca

Causa: Erro raro de memória ou script.

Solução: Recarregue a página (F5). Infelizmente o progresso daquele simulado será perdido, mas o sistema voltará ao normal.

Problemas na Biblioteca

Simulados sumiram

Causa: Limpeza de cache do navegador ou uso de aba anônima.

Prevenção: Sempre exporte seus simulados importantes para o computador (Botão Download na biblioteca).

Não salva mais nada (Limite Atingido)

Solução: O limite é de 10 simulados. Exclua os antigos para liberar espaço. O sistema avisará quando chegar no limite.
